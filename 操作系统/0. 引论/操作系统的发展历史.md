# 随堂笔记

## 计算机操纵系统历史

### 1960

程序进入死循环,为什么系统没有陷入死循环?

病人回来复诊,告诉医生是回来复诊的(cli),医生结束原来的进程,终止循环.等这个病人复诊完成后再执行循环

segmentation fault(错误,跟医生说,对应操作优先执行)
